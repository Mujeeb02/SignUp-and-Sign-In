## Setup

### Frontend Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/Mujeeb02/SignUp-and-Sign-In/frontend.git
   ```

2. Navigate to the frontend directory:

   ```bash
   cd frontend
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm start
   ```
